# Backend Technical Assessment
Developed on NodeJS v20.17.0 with MySQL for database

To run the application, run in command prompt
`npm install` and `npm run start`

To manually populate the database, you can run the file <b>INSERT_CAFE.sql</b> followed by <b>INSERT_EMPLOYEE.sql</b> in the database

# Frontend Technical Assessment
Developed on ReactJS with AG Grid and Ant Design

To build the application, use
`npm install` and `npm run build`
